Exemplo de Lista de Descrição

TCP
O Protocolo de Controle de Transmissão (TCP) é um método (protocolo) usado junto com o Protocolo de Internet (IP) para enviar dados na forma de unidade de mensagem, chamadas de pacotes, entre computadores pela Internet.
IP
O Protocolo de Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para outro na Internet. Cada computador na Internet é identificado de forma única por um endereço IP.
FTP
O Protocolo de Tranferência de Arquivos (FTP) é um protocolo usado para trocar arquivos entre computadores na Internet.
HTTP
O Protocolo de Transferência de Hipertexto (HTTP) é o protocolo usado para trocar texto, imagens gráficas, som, vídeo e outros arquivos multimídia na Web.
